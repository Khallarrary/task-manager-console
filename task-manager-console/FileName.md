# Task Manager (Console)

Projeto de prática em **C#** desenvolvido para treinar conceitos fundamentais de backend antes da construção de APIs.

## Objetivo

Praticar a lógica de um CRUD completo e aplicar conceitos comuns em aplicações backend sem depender de frameworks.

## Conceitos praticados

- CRUD (Create, Read, Update, Delete)
- Service Layer
- Repository Pattern
- Persistência de dados em JSON
- LINQ
- Separação de responsabilidades

## Estrutura do projeto

Entities
└── TaskItem

Repositories
├── ITaskRepository
└── JsonTaskRepository

Services
└── TaskServices

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Buscar tarefa por ID
- Marcar tarefa como concluída
- Deletar tarefa

## Tecnologias

- C#
- .NET
- JSON para persistência de dados

## Observação

Este projeto faz parte de uma série de exercícios voltados ao aprendizado de desenvolvimento backend e preparação para a construção de APIs com ASP.NET.
